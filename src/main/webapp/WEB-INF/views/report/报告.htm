2.4 课程学习结果分析

2.4.1 各部分得分情况

本门课程的成绩由观看视频、作业、测验和讨论四部分组成，四部分的占比

为60:10:15:15，即观看视频总分60 分，测验总分15 分，讨论总分15 分，作

业总分10 分。实际参与课程学习的学生各部分的得分与总分的情况如下图。

由图中可以看出，实际参与学习的学生总成绩平均分为79.3 分，其中在观

看视频方面的表现最好，平均已得到大概91.83%的得分；在讨论方面的表现次

之，平均得分在79.3%左右；再次是在测验方面的表现，平均得到大概62%的

得分；由于作业的提交情况不理想，在作业方面的平均得分较低。
